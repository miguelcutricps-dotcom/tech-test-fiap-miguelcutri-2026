> **Teste** **técnico** **de** **Front-end** **–** **FIAP**

**Desafio**

Você foi selecionado para desenvolver uma Landing Page que acompanha a
identidade visual da FIAP, e alguns componentes que criamos na nossa
rotina de trabalho.

A equipe criou um layout no Figma a ser seguido, contendo as resoluções
desktop (**1920x1080**) e mobile (**414x736**).

**Tecnologias** **obrigatórias**

> • NextJS
>
> • TypeScript • SCSS

As demais dependências para desenvolver a página ficam a seu critério,
mas **tente** **usar** **a** **menor** **quantidade** **possível**.

Utilizamos **yarn** como gerenciador de pacotes, mas pode usar npm caso
prefira.

**Orientações** **para** **entrega** **do** **projeto**

> • Fica ao seu critério como será o layout das resoluções de tablet.
>
> • Adicionar toda a orientação necessária para a instalação da página e
> as dependências usadas em um arquivo chamado **README.md**. Se quiser,
> pode adicionar demonstrações da página em fotos/vídeos dentro do
> arquivo.
>
> • Colocar o projeto no **GitHub** em um repositório **público** e
> mandar um e-mail para **<u>liderancadev@fiap.com.br</u>** com o link
> do projeto.
